Info:
-----

Die Mpeg2Schnitt MOD Extension ist eine Erweiterung des Programms Mpeg2Schnitt von
Martin Dienert (www.mdienert.de/mpeg2schnitt/).
Die MOD Extension erlaubt die Integration von ProjectX
(ProjectX - a free Java based demux utility -> sourceforge.net/projects/project-x/)
im Mpeg2Schnitt Programm.
Die vorhandenen originalen Mpeg2Schnitt Funktionen bleiben unverändert.

Vorraussetzungen:
-----------------

Um eine Mpeg2Schnitt "MOD" Version kompilieren zu können, sind folgende Vorraussetzungen nötig:

1.) Mpeg2Schnitt Quellcode
Die MOD Extension wurde auf Basis der Version 0.87 des Mpeg2Schnitt Quellcodes entwickelt.
Prinzipiell sollte es auch ohne Probleme möglich sein, die MOD Extension in andere Versionsstände
zu integrieren.

2.) Mpeg2Schnitt MOD Extension Quellcode
Zusätzlich zur Mpeg2Schnitt Quellcode Basis wird der MOD Extension Quellcode benötigt, der
in den Mpeg2Schnitt Quellcode Ordner kopiert werden muß.

3.) JEDI Visual Component Library
Die MOD Extension verwendet einige Komponenten aus der "JEDI Visual Component Library", die
auf der Homepage http://homepages.borland.com/jedi/jvcl/ unter der Mozilla Public License
verfügbar ist.



Integration der MOD Extension in Mpeg2Schnitt:
----------------------------------------------

Die MOD Extension kann auf 2 Arten in den Mpeg2Schnitt Quellcode integriert werden:

1.) Patch
Mit der "Patcher.exe" kann man automatisiert die benötigten Mpeg2Schnitt Quellcode
Änderungen vornehmen lassen (siehe "MOD_Patcher_Anleitung.txt"):

2.) Manuell
Da nur wenige Zeilen im original Mpeg2Schnitt Quellcode geändert/ergänzt werden müssen,
kann man die Änderungen auch selber vornehmen (siehe "MOD_Einbauanleitung.txt").

Es reicht aus, nur eine der beiden aufgeführten Methoden durchzuführen.
